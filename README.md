# Quarterly Sales Analyzer

Python program that processes and analyzes sales records from multiple store branches over a 4-month period, using matrices to organize data and calculate key performance indicators.

## Features
- Records sales from 4 stores over 4 months in a matrix structure
- Calculates total and average sales for the period
- Classifies sales against a reference threshold ($2000)
- Displays results in a formatted report

## Technologies
- Python 3

## How to run
```bash
python quarterly_sales_analyzer.py
```
